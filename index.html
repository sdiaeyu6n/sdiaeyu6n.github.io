Diffusion Model Enhancement for Sketch Generation via VQA Feedback
Sieun Choi
Under review   /   Patent application in progress

A pixel-based sketch generation framework that adapts Stable Diffusion to abstract, human-drawn, instance-level sketches. We introduce a VQA-based reward to improve semantic alignment with textual prompts and release an instance-level sketch dataset with captions and QA pairs.

ML-Based Prediction of CBD Stones for ERCP decision-making
Sieun Choi
Preprint

Decision-tree stacking ensemble with feature selection and SMOTE augmentation, developed with the Dongguk University Medical Center to assist emergency-department ERCP decisions; ongoing follow-up on imaging-based CBD diameter measurement.

Dynamic Access Management of Malicious Packets with Reinforcement Learning
Sieun Choi

PPO for a base block/unblock policy, followed by SAC fine-tuning to optimize blocking time and risk; a step toward instance-adaptive, learning-based cybersecurity policies.

Mobile Data Loss Prevention for Real-Time Personal Information Monitoring
Sieun Choi · Undergraduate Capstone (2022)

Android app that detects PII leakage from mobile traffic via packet-capture API; regex-based matching against a sensitive-patterns database with a designed schema.
